## the basic ones

- I 
  - Integer
- F 
  - Float
- N 
  - Numeric, has support for currencies
  - can e.g. insert 10 EUR, 10 USD, 10k USD
- S 
  - string
- B 
  - boolean
  - true,True,1 are all considered to be True
